Objective 1: Set Up the Project

    Create a new Python file for your project.
    Import the necessary modules.

Objective 2: Start and End a Study Session

    Create functions to start and end a study session.
    Use the datetime module to record the start and end times.

Objective 3: Calculate the Duration

    Calculate the duration of the study session.
    Display the duration in a user-friendly format.

Objective 4: Store and Retrieve Sessions

    Store the start and end times of each session in a list or file.
    Retrieve and display past study sessions.

Objective 5: User Interface

    Create a simple text-based interface to interact with the time tracker.
    Allow the user to start, end, and view study sessions.
